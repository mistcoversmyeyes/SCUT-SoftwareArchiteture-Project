## 1. 项目整合管理

### 1.1 项目章程

| 项目要素 | 内容 |
|---------|------|

### 4.2 挣值管理（EVM）跟踪

每周更新挣值数据：

```
公式：
PV（计划价值） = 总预算 × 计划完成百分比
AC（实际成本） = 实际工作人时
EV（挣值）     = 总预算 × 实际完成百分比

CPI（成本绩效指数） = EV / AC
SPI（进度绩效指数） = EV / PV

目标：CPI > 0.95, SPI > 0.90
```

### 8.2 风险应对计划

#### 🔴 R1: 模型转换/集成失败

**应对策略**：规避 + 转移
- **行动1**（Day 0）：立即测试PaddleOCR
- **行动2**（Day 0）：准备备选方案
  - Plan A: PaddleOCR直接使用（推荐）
  - Plan B: HuggingFace ONNX模型
  - Plan C: Tesseract OCR降级
- **触发条件**：Day 1未成功集成
- **责任人**：开发者B

#### 🔴 R2: 性能无法达标

**应对策略**：减轻 + 接受
- **行动1**：Day 5性能基准测试
- **行动2**：优化措施
  - 模型量化（INT8）
  - 批处理优化
  - GPU加速
  - 降分辨率
- **行动3**：调整为"分析影响因素"而非"保证<2秒"
