## 📅 Day 8-14 执行计划（Go微服务升级）⭐

### Day 8-9: Protobuf协议定义

**共享协议**（开发者A+B）：
```protobuf
// shared/proto/ocr_service.proto
syntax = "proto3";

package ocr;

service OCRService {
  rpc ProcessOCR(OCRRequest) returns (OCRResponse);
}

message OCRRequest {
  string request_id = 1;
  bytes image_data = 2;
  repeated BoundingBox detections = 3;
}

message OCRResponse {
  string request_id = 1;
  repeated TextRegion results = 2;
  PerformanceMetrics metrics = 3;
}
```

**编译**：
```bash
# Python
python -m grpc_tools.protoc -I./shared/proto \
  --python_out=./python-mvp/app/proto \
  --grpc_python_out=./python-mvp/app/proto \
  shared/proto/ocr_service.proto

# Go
protoc -I./shared/proto \
  --go_out=./go-gateway/pkg/proto \
  --go-grpc_out=./go-gateway/pkg/proto \
  shared/proto/ocr_service.proto
```
